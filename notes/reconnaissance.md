# 🕵️ Reconnaissance

Reconnaissance is the initial phase of ethical hacking where attackers or testers collect as much information as possible about the target.

## 🔍 Types of Reconnaissance

### 1. Passive Reconnaissance
- Does not directly interact with the target.
- Uses sources like: WHOIS, DNS records, Google Dorking, social media.

### 2. Active Reconnaissance
- Involves direct interaction with the target.
- Examples: ping sweeps, port scanning, banner grabbing.

## 🛠️ Tools

- `theHarvester` – Collect emails, subdomains, hosts
- `Shodan` – Internet-connected device search
- `Recon-ng` – OSINT framework
- `Maltego` – Relationship mapping and link analysis

## ⚠️ Ethical Notice

Recon should only be performed with proper authorization.
